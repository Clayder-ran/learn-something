[git地址](https://github.com/nvm-sh/nvm)

### 定位
* 管理Node和npm版本的工具

### 安装(MacOS)
* `curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash`


### 查找版本
* `nvm ls-remote`: 列出远程Node版本
* `nvm ls`: 已经下载的版本


### 安装Node
* `nvm install 版本号`: 安装; eg `nvm install v13.6.0`

### 切换
* `nvm use 版本号`

### 设置默认版本
* `nvm alias default 版本号`




最好先卸载本机的 node;

* 完成后, 测试一下`nvm -v`;
