### 动态路由
* `:xxxxx`: 冒号开头标识的是动态路由;
```
/* 路由.js */
export default new VueRouter({
    routes: [{
        path: "/user/:name/:id",
        component: User,
    }],
})

/* 组件.js */
<template>
    <div>
        <p>{{$route.params.name}}</p>
        <p>{{$route.params.id}}</p>
    </div>
</template>

<script>
...
    this.$route.params.name  // 读取路径中匹配的值
...
</script>
```
